# Stonez — Personal Portfolio

Source for my personal portfolio site, live at [tammylstonez.netlify.app](https://tammylstonez.netlify.app).

Built with plain HTML, CSS, and JavaScript — no framework, no build step.

## Sections

- **Home** — intro and quick links
- **About** — background and approach to solving problems through web tools (e.g. [EQConvert](https://github.com/TammyLStonez/EQConvert))
- **Services** — Graphic Design, Fashion Design, Web Development, Video Editing
- **Portfolio** — filterable project showcase across design, development, video, and fashion work
- **Contact** — message form that delivers via WhatsApp

## Tech Stack

- HTML5 / CSS3 / vanilla JavaScript
- Netlify (hosting + deployment)

## Running Locally

```bash
git clone https://github.com/TammyLStonez/Stonez.git
cd Stonez
```

No build step or dependencies — open `index.html` directly in a browser, or serve the folder with any static server (e.g. the VS Code "Live Server" extension).

## Deployment

Auto-deploys to Netlify on every push to `main`.
