About Me


I'm Livingstone, a Front-end developer who builds tools that makes work easier.I don't just write code, I solve problems. When an Educational Organisation needed a faster way to convert LaTeX equations in their transcribed documents to Unicode, I built EQConvert, a web app that allows users upload files, converts every equation automatically with real-time logs, and hands back a clean document. What used to be hours of manual formatting now takes seconds. That's the kind of work I do: identify the friction, build the fix, and measure it by the time and stress it saves. If you have a workflow problem that a well-built web tool could solve, I'd like to hear about it.
